Free Inference

Moteur d'inference LLM open-source, local et souverain. Pour les entreprises et développeurs qui veulent garder le contrôle de leurs données.

GitHub Documentation Tester le Chat

GPU Local

Inference sur GPU NVIDIA avec vLLM. Optimisé pour 12 Go VRAM. Pas de dépendance à des API externes.

Gateway API

LiteLLM Proxy avec authentification, budgets et rate limiting. Compatible OpenAI API.

Multimodal

OCR, Speech-to-Text et embeddings en local. Traitez images, audio et texte sans quitter votre machine.

Open Source

MIT License. Déployez chez vous, modifiez, contribuez. Aucune vendor lock-in.
